# secure-password-intelligencee
A cybersecurity-focused Java application to analyze password strength using scoring, entropy, and secure hashing (SHA-256), with CLI and GUI support.
# 🔐 Secure Password Intelligence System

## 🎯 Problem Statement

Weak passwords are one of the leading causes of security breaches.
This project analyzes password strength and promotes secure password practices.

---

## 🚀 Features

* Password strength scoring
* Entropy calculation (advanced security)
* Secure hashing using SHA-256
* CLI (Command Line Interface)
* JavaFX GUI (User Interface)
* Input validation and error handling
* Test case coverage
* Cloud logging simulation

---

## 🛠️ Tech Stack

* Java
* JavaFX
* Cryptography (SHA-256)
